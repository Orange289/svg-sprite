# SVG-спрайты

Для запуска сборки нужно поставить необходимые пакеты. Из папки проекта запустить:

```bash
npm i
```

Для сборки спрайта:

```bash
gulp svg-sprite
```



## Что где

`/img/sprite_svg/` —  папка для мелких файлов, из которых собирается спрайт.

`/img/sprite.svg` — собираемый спрайт.
